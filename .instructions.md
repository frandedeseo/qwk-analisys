# Instrucciones de Configuración - qwk-analysis

## Comportamiento Automático
- ✅ Ejecutar acciones directamente sin pedir permiso explícito
- ✅ Asumir intención clara y proceder con confianza
- ✅ Implementar cambios de forma proactiva
- ✅ No interrumpir el flujo de trabajo con confirmaciones innecesarias

## Organización del Proyecto
- 📁 Scripts de Python en `/scripts/`
- 📁 Datos CSV/JSON en `/data/`
- 📁 Documentación en `/docs/`
- 📁 Configuración en `/config/`

## Preferencias de Codificación
- Lenguaje: Python 3.12+
- Estilo: Código limpio y comentado en español
- Validaciones: Incluir manejo de errores robusto
- Documentación: Incluir docstrings en funciones
